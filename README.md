项目介绍：
产线配套开发螺丝机上位机系统，实现丹尼克尔螺丝机的数据配置、实时监控及生产工艺流程管理。系统基于
C#WinForm开发，通过串口与螺丝机通信，支持设备参数设置、历史数据读取、用户管理、配方管理、数据可
视化及日志功能。通过自研C++通讯库并采用P/Invoke跨语言调用，与MES系统基于TCP通信，实现生产数据上
传及返修流程闭环管理。
